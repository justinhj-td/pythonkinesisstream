# Read a kinesis stream

## Setup

```
pyenv install 3.9.5
export PYENV_ROOT="$HOME/.pyenv"
export PATH="$PYENV_ROOT/bin:$PATH"
eval "$(pyenv init --path)"
pyenv local 3.9.5
pip install --upgrade pip
pip install -r requirements.txt
```

## Running

Edit stream.py and enter your stream name
`python stream.py`
